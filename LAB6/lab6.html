  1. ¿Cuáles son las diferencias entre los posibles valores de la propiedad position?
    • Static, es el valor que toma un elemento por defecto para posicionar y respetará el flujo normal de la pag.
    • Relative, . Se podría decir que estamos posicionando un elemento tomando como referencia el flujo normal
    • Absolute, no estará dentro del flujo normal de la página y tomará como referencia la ventana del navegador o el elemento posicionado
    • Fixed, oman como referencia la ventana del navegador y no respetan el tener un contenedor padre que esté posicionado.
  2. ¿Cuáles son los valores estándar para la propiedad visibility?
    • Visible, hidden, collapse inherit
  3. ¿Qué es el zIndex y para qué sirve?
    • La propiedad CSS z-index indica el z-order de un elemento y sus descendientes. Cuando varios elementos se superponen, z-order determina cuales cubren a otros. Un elemento con mayor z-index generalmente cubre a otro menor.
